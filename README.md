# Calculator-GUI

## Описание
This calculator is built using Python and Tkinter. It has buttons that display input on the screen when pressed.

## Возможности
- There are buttons that display numbers and signs on the screen when pressed
- The signs include +, -, *, /, =, C and digits from 0 to 9.
- Pressing C clears all input from the display line.

## Запуск
python main.py
